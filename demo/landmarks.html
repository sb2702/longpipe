Landmark probe — full GPU-resident chain, up to 4 faces

The whole path runs on the GPU with zero per-frame readback: TierModel matting pass → FaceHeatmapNet → FaceBoxesFromHeatmaps (multi-face decode, compute shader) → CropResample (256² crop + ImageNet norm) → LandmarkNet (478 pts) → LandmarkOverlay, whose vertex shader reads the landmark tensor directly (vertex pulling) and transforms crop → canvas via the box tensor. The optional stats line reads the box tensor every 30 frames — untick to make the loop literally readback-free.

multi-face: one decode emits a 1×4×4 box tensor; each slot drives its own crop + landmark net + overlay pass, differing only by a slot index — so there is no CPU-side branch on face count anywhere, and cost is a fixed 4 landmark runs per frame regardless of how many people are in shot. Empty slots carry score 0, which the overlay's vertex shader clips away. Each face gets its own dot color. Off = the single-face FaceBoxFromHeatmaps production path, for A/B. Toggling restarts the chain (K crops and landmark nets are built at startup).
