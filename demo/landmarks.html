Landmark probe — full GPU-resident chain

The whole path runs on the GPU with zero per-frame readback: TierModel matting pass → FaceHeatmapNet → FaceBoxFromHeatmaps (soft-argmax box, compute shader) → CropResample (256² crop + ImageNet norm) → LandmarkNet (478 pts) → LandmarkOverlay, whose vertex shader reads the landmark tensor directly (vertex pulling) and transforms crop → canvas via the box tensor. The optional stats line reads the 16-byte box tensor every 30 frames — untick to make the loop literally readback-free.
